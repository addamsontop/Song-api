Introduction

Bienvenue dans la documentation de l'API Musicale pour bots Discord créée par addamsontop. Cette API vous permet d'intégrer facilement des fonctionnalités musicales dans vos bots Discord.

Authentification

Toutes les requêtes doivent inclure un token d'API dans l'en-tête x-api-token.

Rejoindre un salon vocal

POST /join

Permet au bot de rejoindre un salon vocal.

JavaScript
BDFD

fetch('https://addams-song.vercel.app/api/join', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-token': 'votre_token_secret'
  },
  body: JSON.stringify({
    guildId: 'ID_DU_SERVEUR',
    voiceChannelId: 'ID_DU_SALON_VOCAL'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erreur:', error));
                

$httpPost[https://addams-song.vercel.app/api/join;{"guildId":"ID_DU_SERVEUR","voiceChannelId":"ID_DU_SALON_VOCAL"}]
$httpHeader[x-api-token: votre_token_secret]
$nomention Le bot a rejoint le salon vocal.

Jouer une chanson

POST /play

Le bot joue une chanson dans le salon vocal.

BDFD

  method: 'POST',

  },
  body: JSON.stringify({
    guildId: 'ID_DU_SERVEUR',
    voiceChannelId: 'ID_DU_SALON_VOCAL',
    songUrl: 'URL_DE_LA_CHANSON'
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Erreur:', error));
                

$httpPost[https://addams-song.vercel.app/api/play;{"guildId":"ID_DU_SERVEUR","voiceChannelId":"ID_DU_SALON_VOCAL","songUrl":"URL_DE_LA_CHANSON"}]

$nomention Lecture de la chanson en cours.
